## Quick Deploy to Railway

1. **Create GitHub repo:**
   ```bash
   cd rrk-blog-railway-deploy
   git init
   git add .
   git commit -m "RRK Law blog with BlogPosting schema for AI indexing"
   ```

2. **Push to GitHub:**
   - Create new repo on GitHub: `rrk-blog-articles`
   - Push this code

3. **Deploy on Railway:**
   - Go to railway.app
   - New Project → Deploy from GitHub
   - Select `rrk-blog-articles` repo
   - Railway auto-detects Dockerfile and deploys

4. **Add custom domain:**
   - In Railway project → Settings → Domains
   - Add: `blog.rrklawoffice.com`
   - Copy CNAME target
   - Add in Cloudflare DNS (proxy OFF)
